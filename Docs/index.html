Introdução HTML e CSS

HTML (HyperText Markup Language) e CSS (Cascading Style Sheets) são as duas linguagens essenciais que formam a base da criação de páginas da web modernas e atraentes. Juntas, elas desempenham papéis complementares e cruciais na construção e no design de sites e aplicativos web.

O HTML é a linguagem de marcação que define a estrutura e o conteúdo de uma página da web. Ele utiliza elementos como < p> para parágrafos, < h1> a < h6> para cabeçalhos, < a> para links e muitos outros para criar a estrutura básica de um documento web. O HTML também permite incorporar elementos multimídia, como imagens, áudio e vídeo, bem como criar formulários interativos para coletar informações dos usuários. É a espinha dorsal de qualquer página da web, fornecendo a estrutura semântica que os navegadores e motores de busca entendem.

Por outro lado, o CSS é responsável pelo estilo e pela apresentação visual de uma página da web. Com o CSS, você pode controlar a aparência de elementos HTML, como cores, fontes, margens, tamanhos e posicionamento. Isso permite criar layouts atrativos e garantir que o conteúdo seja apresentado de forma atraente em diferentes dispositivos e tamanhos de tela. O CSS também permite a criação de animações e efeitos visuais, tornando as páginas web mais interativas e dinâmicas.

Tags HTML

Tipos de tag:

    (head): Contém informações sobre o documento, como metadados, títulos da página, e links para arquivos externos, como folhas de estilo CSS.

    (title): Define o título da página, que é exibido na barra de título do navegador.

    (link): Utilizado para vincular o documento HTML a folhas de estilo CSS externas.

    (style): Permite a inclusão de estilos CSS diretamente no documento.

    (script): Usado para incluir código JavaScript no documento ou para fazer referência a arquivos de script externos.

    (body): Contém o conteúdo visível da página, como texto, imagens e elementos interativos.

    (h1), (h2), (h3), ... (h6): Definem cabeçalhos de seção com diferentes níveis de importância.

    (p): Cria parágrafos de texto.

    (ul) e (ol): Criam listas não ordenadas (com marcadores) e ordenadas (com números ou letras).

    (li): Define itens de lista em listas (ul) ou (ol).

    (div): Usado para criar blocos de conteúdo genéricos e facilitar a estilização com CSS.

    (span): Similar ao (div), mas aplicado a elementos inline para estilização mais específica.

    (audio) e (video): Permitem incorporar áudio e vídeo em páginas da web.

    (section) em HTML é um elemento de marcação usado para agrupar e identificar conteúdo relacionado em uma página da web.
